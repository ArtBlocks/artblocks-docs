# Creative Offerings

Art Blocks is a platform for on-chain generative art. Artists write algorithms that live permanently on the blockchain, and collectors mint unique, deterministic artworks from those algorithms. Since 2020, Art Blocks has hosted hundreds of projects by artists from around the world.

---

## Art Blocks Studio

**Art Blocks Studio** is the primary self-service path for releasing generative art on Art Blocks. Studio artists have full control over their project configuration — scripts, minting mechanics, edition size, payment splits — through the [Art Blocks Creator Dashboard](https://create.artblocks.io/).

Studio projects are released on the Art Blocks platform at [artblocks.io](https://artblocks.io), alongside Curated and historical projects.

### What Studio Provides

- A V3 Engine core contract deployment
- Access to the Art Blocks Creator Dashboard for project setup and management
- Sepolia testnet staging environment for QA and review
- Integration with the Art Blocks rendering infrastructure (token API, generator, media proxy)
- Full Art Blocks Minter Suite (Dutch Auctions, allowlists, RAM, SEA, and more)
- ERC-2981 royalties via 0xSplits
- Listing on artblocks.io with full platform support

## How to Apply

Art Blocks curates Studio artists carefully. The acceptance rate is approximately 10%. Art Blocks looks for artists who demonstrate:

- **Innovation** — work that pushes the boundaries of what generative code can express
- **Technical proficiency** — scripts that are robust, well-crafted, and ready for on-chain deployment
- **Conceptual depth** — a clear artistic vision and thoughtful engagement with the medium

**Apply at [artblocks.io/apply](https://artblocks.io/apply)**

Applications are reviewed on a rolling basis. You'll receive feedback regardless of the outcome.

### Contact

- Applications: [apply@artblocks.io](mailto:apply@artblocks.io)
- Technical questions: `#artist-tech` channel on the [Art Blocks Discord](https://discord.gg/artblocks)
- General questions: `#artist-general` channel on Discord

## Art Blocks Engine

**Art Blocks Engine** is a white-label solution that allows organizations and platforms to deploy their own generative art infrastructure using Art Blocks' smart contracts and rendering stack.

Engine partners own their contracts and release as many projects as they like. Engine is used by galleries, media companies, game studios, fashion brands, and other organizations interested in generative digital collectibles.

**Interested in Engine?** Contact [info@artblocks.io](mailto:info@artblocks.io).

See the [Engine Partners section](/creator-onboarding/engine-partners/what-is-engine/) for more details.
